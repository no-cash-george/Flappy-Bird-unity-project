# Flappy Bird Unity Project 🐦

A faithful recreation of the iconic Flappy Bird game, developed as my introductory project into **Game Development** and **C# scripting** within the Unity Engine.

## 🚀 Overview
This project served as the foundation for learning the Unity lifecycle, physics-based movement, and 2D sprite management. It focuses on simple yet addictive gameplay mechanics and robust collision systems of the famous game Flappy Bird.

## 🛠️ Technical Features
* **Physics-Based Gameplay:** Precise implementation of gravity and upward force (flapping) using Unity's `Rigidbody2D`.
* **Procedural Obstacle Spawning:** A system that generates pipes at random heights to ensure a unique experience in every playthrough.
* **Collision Logic:** Detailed trigger handling for scoring and lethal collisions with obstacles or the ground.
* **UI Management:** Real-time score tracking and a dynamic "Game Over" state with restart functionality.

## 💻 Technologies Used
* **Engine:** Unity
* **Language:** C#
* **Assets:** 2D Sprites and Physics Materials

## 🎮 How to Play
1. Clone the repository.
2. Open with Unity Hub.
3. **Controls:**
    * `Space` or `Left Mouse Click`: Flap wings to gain altitude.
    * Avoid the pipes and don't hit the ground!

## 📈 Learning Outcomes
This first project allowed me to master:
* The **Input System** in Unity.
* Working with **Prefabs** for dynamic object instantiation.
* Basic **Game State Management** (Start, Playing, Game Over).
* Understanding the importance of **DeltaTime** for frame-independent movement.
